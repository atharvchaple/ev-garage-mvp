 "EV Garage Management MVP"

A Flask-based EV garage management system.

1) Features:-
 Customer Management
 Vehicle Tracking
 Battery History
 Service Records
 Invoice Generation

2) Tech Stack:-
 Python
 Flask
 SQLite
 HTML/CSS

3) How to Run:-
1. Install dependencies:
   pip install flask flask-sqlalchemy
2. Run:
   python app.py
